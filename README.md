# yandex-archive-helpers
Скрипты для работы с сервисом Яндекс Поиск по архивам

## parse-page.js
Скрипт для запуска из консоли браузера, который отправляет данные найденных дел на ваш обработчик.
Запускаем скрипт со страницы списка записей (результатов поиска).

```"АДРЕС ВАШЕГО ОБРАБОТЧИКА"``` заменить на url, по которому будет работать обработчик полученных данных.
